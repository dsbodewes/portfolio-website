PROJECTS

Math Physualizer


Dit project bestaat uit meerdere kleine projecten waarin veel wiskunde voor komt, dit doe ik in Raylib C++.

Hier ben ik aan begonnen omdat ik mijn kennis in wiskunde wilde verbreden, dit helpt mij om wiskunde beter te kunnen visualizeren.

Momenteel bestaat het project uit een Spaceship, een Pendulum, een Dubbele Pendulum en een Boid System.

Zie hier repository op GitHub

Project Architectuur: PEEK


Tijdens dit project hebben we de opdracht gekregen om aan een game te werken met een thema waar je je aan moest houden. In ons geval was het thema "Gothiek", dus hadden we besloten om een horror game te maken waarbij je uit de stad moet onstsnappen door alle generatoren te repareren.

In dit project was ik verantwoordelijk voor de enemy AI in de game, ook heb ik een aantal modellen in de game gemaakt met Blender.

Het project is gemaakt in Unity en C#.

Zie hier repository op GitHub

Global Game Jam 2024: Tweakin'


Tweakin' is de Global Gamejam Groningen Curveball Award winnaar van 2024.

Het thema was "Make me laugh" en wij hadden dus besloten om dat te gaan doen op een andere manier.

De game is gemaakt met Unity en C#.

Download de game op Itch.io

Project Studio: Traffic System

Tijdens dit project heb ik gewerkt aan een verkeersysteem, dat was een onderdeel van de Projection Mapping. Hiervoor heb ik een Splines en Junctions package gekregen om dit voor elkaar te krijgen. Het was een interessant project om aan te werken, omdat ik nog nooit met Splines en Junctions had gewerkt.

Gemaakt met: Unity en C#.

Interfaces: Arcane Warfield

Voor dit project hadden we besloten om een battle simulator te maken, het uiteindelijke doel was eigenlijk om deze game compatable te krijgen met een zelf gemaakt toestel, maar dat is helaas niet gelukt. Ik heb gewerkt aan de placement van de troepen en aan 1 van de tovenaars.

Gemaakt met: Unity, C#, Arduino IDE en C++.

Serious App: Spendatie Sensatie

Dit project was in samenwerking met de Groningse Krediet Bank. Het doel van het project was om jongeren beter bewust te laten zijn van in-game aankopen. Ik heb gewerkt aan de werkingen van de UI van de app.

Gemaakt met: Unity en C#.

RT2D/Framework

Dit was een project waarbij het doel was om OOP te leren. Eerst heb ik een simpel spel gemaakt in het framework: RT2D. Hier heb ik een simpele platformer gemaakt, met een character die kan springen en bewegen. Daarna heb ik mijn eigen framework gemaakt om spellen in te maken.

Gemaakt met: RT2D, RayLib en C++

Start-Ups: Daydream Studio

Voor dit project hebben we als groep een fictioneel game bedrijf ingericht en daar een website voor gemaakt. Hierbij hebben we ook een spel concept bedacht en een aantal renders gemaakt van de game. De game heet LITH maar bestaat niet sinds dit een concept was voor ons game bedrijf.

Gemaakt met: HTML en CSS.
